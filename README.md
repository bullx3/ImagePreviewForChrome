# ImagePreviewForChrome
chrome 拡張. サイトの画像を漫画ビュアー風に表示

## Descrition

chrome拡張機能でサイトの画像を漫画ビュアー風に表示する  
画像は左右二分割で表示（右 -> 左）  
元の画面に戻すにはブラウザの”戻す”かポップアップの"戻す"ボタン
上下左右キーでページ切り替え可
シークレットモードでも利用可
